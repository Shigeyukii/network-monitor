# Network Monitor

ネットワーク監視アプリケーション

## 概要

このアプリは、ネットワーク上のデバイスを監視し、Ping応答やTCPポートの状態を追跡するiOSアプリケーションです。

## 主な機能

- デバイスのPing監視
- TCPポートの監視
- バックグラウンドでの定期的な監視
- 通知機能
- SwiftDataを使用したデータ永続化

## 技術スタック

- SwiftUI
- SwiftData
- Swift Concurrency (async/await)
- BackgroundTasks framework
- UserNotifications framework

## 必要環境

- iOS 17.0以降
- Xcode 15.0以降

## ビルド方法

1. Xcodeでプロジェクトを開く
2. 必要に応じて開発チームを設定
3. ビルド & 実行
